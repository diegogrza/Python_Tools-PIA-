Luis¶

Este script va a ligando a un script anterior llamado mail,el cual saca varios correos de una pagina, este script lo que hace es que va a empezar a leer linea por linea todos los correos que previamente se guardaron en un documento y empezara a enviarles un correo eectronico. Para que este script funcione ocupamos:

# Importamos librerías (logging, smtplib, ssl, mimetypes) # Importamos los módulos necesarios (MIMEMultipart) # Creamos objeto Multipart, quien será el recipiente que enviaremos (Todos los correos en el archivo) # Autenticamos (Entramos a nuestro correo) # Enviamos # Cerramos conexión
