QR¶

Al usar el parámetro “-encode” se espera que se escriba una ruta de un archivo > Ejemplo de uso: pia.py -tool “qr” -encode “ruta del archivo txt” Al usar el parámetro “-decode” se espera que se escriba la ruta de un archivo t> Se generará un archivo wrlgen.txt en la carpeta URLS con las urls de cada image> Ejemplo de uso: pia.py -tool “qr” -decode “.URLSnqr.txt”

Codigo QR¶

Encrip im¶

El script con el parámetro -encriptar se deberá poner la ruta en la que se encuentra una imagen en la que se quiera ingrear un mensaje cifrado, para para el mensaje cifrado se pone el parámetro -msg en el que se pone el msj que se quiere cifrar, con el parámeto -nf se debe poner el nombre de la imagen que se quiere genera, como observación en el nombre de la imagen resultante es SIN terminación .jpg o .png, ya que automáticamente genera en formato .png.

Ejemplo de uso: pia.py -tool “encrip im” -encriptar “ruta de la imagen” -msg “Mensaje a cifrar” -nf “nombre de imagen sin terminacion png o jpg”

Al usar el parámetro -desencriptar se deberá especificar la imagen con el menasje cifrado y con el parámetor -key se deberá poner el nombre del archivo con terminación .key que se generó y para terminar con el parámetro -nd se deberá poner el nombre del archivo con terminación txt en el que se imprimirá el mensaje descifrado. Ejemplo de uso pia.py -tool “encrip im” -desencriptar “ruta de la imagen” -key “ruta de la key” -nd “nombre del txt con el mensaje descifrado” Como observación, la imagen que se cifre, la llave y el mensaje descifrado se encontrarán dentro de una carpeta llamada extraer, esta se encuentra en la ruta donde está el script encrip_im.py.

Actualizador¶

El script ProgT.py, se encarga de ejecutar el programa Programador.py desde la aplicacion de python.

Codigo ProgT¶

El script Programador.ps1 se encargara de programar la tarea, ingresando la particion de la aplicacion de python, ejecutando el actualizador.py, despues el codigo utilizara las configuraciones correpsondientes con los parametros y condiciones que se le añadieron, para que al ser ejecutada se registre en el programador de tareas.

Codigo Programador¶

El script Pr.ps1 se encarga de crear una segunda condicion, proveniente de la segunda opcion del condicional de actualizador.py, donde al no cumplirse el dia seleccionado detendra la funcion de actualizacion con el fin de disminuir procesos e interrupciones.

Codigo Pr¶

El script Instalador.ps1 se encarga de crear condiciones escritas en powershell para las situaciones en caso de que al momento de ejecutar el actualizador, la instalacion de las actualizaciones se lleven a cabo en segundo plano solo si se cumple la condicion del dia y la hora.

Codigo Instalador¶

El actualizador.py se encarga de tener el control de los dias en los que se desean ejecutar los programas cargados y escritos en powershell, se importa el modulo datetime que ayudara en el registro del tiempo actual, asi como tambien se usa subprocess para llevar a cabo ejecuciones de programas escritos con powershell. Usando un condicional, es como haremos que el programa solo se ejecute en el dia seleccionado.

Luis¶

Este script va a ligando a un script anterior llamado mail,el cual saca varios correos de una pagina, este script lo que hace es que va a empezar a leer linea por linea todos los correos que previamente se guardaron en un documento y empezara a enviarles un correo eectronico.
